## About

平时会写一些小工具，也会把学习和实践过程整理成文章或笔记。

博客主要放实际折腾过程、工具开发记录和踩坑复盘；知识库更偏系统化整理，主要是 Linux、Shell、Docker、Kubernetes、可观测性、数据库和后端开发这些方向。
