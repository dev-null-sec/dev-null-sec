## Projects

### [ai-board](https://github.com/dev-null-sec/ai-board)

给 AI agent 用的本地项目看板和流程管理工具。

它主要用来记录 AI 开发过程里的任务、scope、状态和验收结果。  
你可以直接让 AI “接手项目”或“把需求进看板”，CLI 负责把这些信息落到项目文件里，方便后续查看、交接和回滚。

### [pents-agent](https://github.com/dev-null-sec/pents-agent)

一套面向授权 Web/API 安全测试的 AI 工作环境。

它不是传统图形化渗透平台，而是给 Claude Code 这类 agent 准备的测试流程、scope 约束、skill、字典、证据记录和报告模板。  
目标是让授权测试过程更可追踪，也让每次测试后的经验可以沉淀下来。

> 只用于授权安全测试。

### [KubeQuest](https://github.com/dev-null-sec/KubeQuest)

一个在浏览器里练习 Kubernetes / kubectl 的模拟器项目。

目前更偏 demo 和实验性质，用来做一些 Kubernetes 命令练习、交互式学习和前端展示上的尝试。
